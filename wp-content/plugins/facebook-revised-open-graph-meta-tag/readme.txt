Adds automatically the meta tags required by Facebook to your pages, posts, homepage, shopp product.

== Description ==
This plugin simply adds the required Facebook Open Graph meta tags on every page with conditions depending if you are on a page, on a post, etc

# Since version 0.3 #
- You now have the ability to choose on Facebook what thumbnail you want to show

*For any comments, questions, problems, suggestions, etc  -- email me@thecodep0et.com



== Installation ==

Simply install with the wordpress backend as any other plugins 

No other steps necessary

== Frequently Asked Questions ==

= What if I have no image ? =

By default, the plugin will add a default image of your choice, you simply need to add in your current theme directory, under a folder called "images", an image called "facebook-default.jpg"
